Ggplot2

Task 1

Create a simple scatterplot using ggplot2 where x axis shows EngineSize, y axis shows MPG.city.

  • Use blue color for your dots.
  • Use size = 2 for the dot sizes.
  • Use xlim and ylim to make sure your x and y axis ranges make sense.
  • Add a title, subtitle, x axis label, y axis label, and a caption using title, subtitle, caption options inside + labs().
ggplot(Cars93, aes(EngineSize, MPG.city)) + geom_point(size=2,col="blue") + xlim(1,6) + ylim(10,50) + labs(title = "The Relationship Between Engine Size and City MPG", subtitle="For 93 Cars on Sale in the USA in 1993", caption = "Figure 1") + xlab("Engine Size") + ylab("City MPG")

Task 2

Recreate the scatterplot in Task 1. Add the following functionality:

  • Use a different color for each car Type.
ggplot(Cars93, aes(EngineSize, MPG.city)) + geom_point(size=2,col=1:length(Cars93$Type)) + xlim(1,6) + ylim(10,50) + labs(title = "The Relationship Between Engine Size and City MPG", subtitle="For 93 Cars on Sale in the USA in 1993", caption = "Figure 2") + xlab("Engine Size") + ylab("City MPG")

Task 3

Recreate the scatterplot in Task 2. Add the following functionality:

  • Fit and overlay a smooth curve.
  • Make the curve line red, and dashed. (Hint linetype is the option you can use to set line style as ‘dashed’)
ggplot(Cars93, aes(EngineSize, MPG.city)) + geom_point(size=2,col=1:length(Cars93$Type)) + xlim(1,6) + ylim(10,50) + labs(title = "The Relationship Between Engine Size and City MPG", subtitle="For 93 Cars on Sale in the USA in 1993", caption = "Figure 3") + xlab("Engine Size") + ylab("City MPG") + geom_smooth(col="red", linetype = "dashed")
## `geom_smooth()` using method = 'loess' and formula = 'y ~ x'

Task 4

Create a barplot showing count of cars by AirBags using ggplot2.

  • geom_bar() is the function that creates a barplot.
  • Use a different color for each bar.
  • Add a title, subtitle, x axis label, y axis label, and a caption using title, subtitle, caption options inside + labs().
ggplot(Cars93) + geom_bar(aes(x=AirBags), col=c(2,3,4), fill=c(2,3,4), stat="count") + labs(title = "The Count of Cars by Type of Air Bag", subtitle="For 93 Cars on Sale in the USA in 1993", caption = "Figure 4") + xlab("Type of Air Bag") + ylab("Count")

Task 5

Recreate the barplot in Task 4 for different car Origin, such that you create one barplot for each Origin category and display all on the same graph.

  • You can have Origin as the “x” variable by setting x=Origin.
  • Use fill=AirBags to get a different color (and bar) by AirBags.
  • in geom_bar(), useposition=‘dodge’` to have the barplots side by side.
ggplot(Cars93) + geom_bar(aes(x=Origin), col=c("lightblue2","lightgreen"), fill=c("lightblue2","lightgreen"), stat="count", position='dodge') + labs(title = "The Count of the Car `Origin` by Type of Air Bag", subtitle="For 93 Cars on Sale in the USA in 1993", caption = "Figure 5") + xlab("Type of Air Bag") + ylab("Count")
